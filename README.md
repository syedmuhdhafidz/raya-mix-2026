# Raya Mix 2026

A festive Raya music web player inspired by a glassmorphism UI concept, rebuilt for 2026.

## Features

- Raya-themed playlist interface
- Search bar for filtering songs
- Functional audio player
- Play, pause, next, previous, and shuffle controls
- Progress bar with seeking
- Curated collectible panel with 3D ketupat mascot
- Responsive glassmorphism layout

## Project Structure

```bash
raya-mix-2026/
├── index.html
├── style.css
├── script.js
└── assets/
    ├── songs/
    ├── covers/
    ├── mascot/
    └── ui/
```

## How to Run

1. Download or clone this repository
2. Open the project folder
3. Launch `index.html` in your browser

## Notes

- Song files are stored in `assets/songs/`
- Cover images are stored in `assets/covers/`
- UI assets and collectibles are stored in `assets/ui/`
- The ketupat mascot is stored in `assets/mascot/`
